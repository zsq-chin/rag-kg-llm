# 引 言

# 0.1 编制的背景及原因

随着辽河油田各区块的不断开发 油气 汽 水井措施逐步成为了增产增注 区块稳产 延缓递减率的重要措施之一 年制定的 $\mathrm { Q / S Y \ L H \ 0 7 2 2 - 2 0 2 2 }$ 采油工艺措施效果评价方法 已经无法满足工作需求 需要进一步修改完善 才能更加科学 合理地对油气 汽 水井措施进行评价

# 编制的目的

标准明确了地质措施和工艺措施范围 规范了各项措施的措施前标定 措施有效期 措施后产量以及效果评价方法等内容 制定了措施有效率 经济有效率 吨油成本等具体计算方法

# 0.3 标准的作用

修改完善的油气 汽 水井措施评价方法能够更加准确的评价措施实施效果 可为生产经营决策提供科学依据 从而提高油气 汽 水井的生产效果

# 不可对比井

主要包括四种情况

) 地层因素 (地质认识、压力异常、地层窜漏)等非措施施工原因造成未完成施工的井,可列为施工成功不可对比井  
b) 地质认识井 新工艺新技术试验井 未达到评价标准的可列为措施有效和经济有效不可对比井  
) 措施有效期内 因套变 汽窜 出砂 地质措施等非本施工原因造成措施有效中断的井 可列为经济有效不可对比井  
) 因自然灾害等不可抗拒因素、不可预测井筒质量等原因未达到评价标准的井,可列为不可对比井。

) 连续生产井出现正常连续生产 天平均日产油 (气)低于措施前日产油标定 (堵水措施出现正常连续生产 天平均含水率高于措施前含水标定);或实施了新措施;

# 5 油气 (汽)水井措施效果评价方法

# 5.1 地质措施效果评价方法

# 5.1.1 大修

措施初期日产油 气 量大于措施前标定

# 5.1.2 侧钻

措施初期日产油 (气)量大于措施前标定。

# 5.1.3 调补层

措施初期日产油 气 量大于措施前标定

# 5.2 工艺措施效果评价方法

# 5.2.1 压裂

施工成功后,符合下列条件之一,判定为措施有效:

a) 措施初期日产油 气 量大于措施前标定  
b) 措施初期日注入量达到配注要求蒸汽吞吐井压后 不超过措施前注汽压力条件下 注入速度提高

# 5.2.2 防砂

施工成功后,符合下列条件之一,判定为措施有效:

) 措施初期日产油 (气)量大于措施前标定;) 连续生产井措施后免修期 $\geqslant 9 0$ 天,且高于油井措施前最后一次因砂检泵 (或其它作业)后生产时间 吞吐井本周期发生因砂检泵次数不高于措施前吞吐周期检泵次数

# 5.2.3 油井解堵

施工成功后,符合下列条件之一,判定为措施有效:

) 措施初期日产油 (气)量大于措施前标定;吞吐井措施后注汽压力低于措施前注汽压力

# 5.2.4 水井解堵

施工成功后 符合下列条件之一 且有效天数大于 天 判定为措施有效

) 施工目的层达到地质配注或措施后日注水量比措施前日注水量提高 $50 \%$ 以上;  
) 解堵目的层段压降曲线趋势明显。

# 5.2.5 堵水

施工成功后 符合下列条件之一 判定为措施有效

措施初期日产油 气 量大于等于措施前标定 平均含水率下降 个百分点及以上) 措施初期日产油 (气)量小于措施前标定,平均含水率下降 个百分点及以上。

# 5.2.6 油井调剖 (封窜)

施工成功后 符合下列条件之一 判定为措施有效

) 措施初期日产油 气 量大于措施前标定  
) 吞吐井措施后注汽压力高于措施前注汽压力或汽窜得到抑制) 吸汽剖面或井温曲线发生明显改变  
d) 措施井综合含水下降 个百分点及以上。

# 5.2.7 水井调剖

施工成功后 恢复正常注水之日起的 个月期间内 累计 天符合下列条件之一时 均视为有效

a) 措施后井组平均日产油高于措施前井组标定产量;  
井组综合含水下降 $2 \%$ 以上 含 $2 \%$ );  
) 在可对比的测试条件下 吸水剖面有明显改善 主要调剖目的层吸水百分率改变 $20 \%$ 以上 或高、低渗透层段相对吸水量改变 $10 \%$ 以上;  
吸水指示曲线上移 启动压力升高 视吸水指数下降  
) 调剖目的层段压降曲线上移或明显变缓。

# 5.2.8 稠油分选注

施工成功后 符合下列条件之一 判定为措施有效

) 措施初期日产油 (气)量大于措施前标定;  
) 吞吐井措施后注汽压力高于措施前注汽压力或汽窜得到抑制) 吸气剖面或井温曲线发生明显改变  
d) 措施井综合含水下降 个百分点及以上

# 5.2.9 多介质复合蒸汽吞吐

施工成功后 措施初期日产油 气 量大于措施前标定 判定为措施有效
